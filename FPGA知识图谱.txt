详细知识分解
================================================================

一、L1 数字电路基础
----------------------------------------------------------------
1.1 组合逻辑
    概念: 输出仅取决于当前输入, 无记忆
    实现: assign 连续赋值, always @(*)
    项目: diag_top.v — assign led_all[0] = cnt[24]

1.2 时序逻辑
    概念: 输出取决于当前输入 + 历史状态, 有记忆(触发器)
    实现: always @(posedge clk)
    项目: run_led.v — 25位计数器

1.3 时钟分频
    概念: 高速时钟 → 计数器 → 低频使能脉冲
    关键: 生成使能信号而非分频时钟(避免新时钟域)
    项目: clock_bd.v — CNT_MAX=25_000_000, 50M→1Hz

1.4 复位策略
    异步复位: always @(posedge clk or negedge rst_n)
    同步释放: POR 上电复位延迟计数器
    项目: alarm_clock_bd.v — POR上电复位

================================================================

二、L2 Verilog 语言
----------------------------------------------------------------
2.1 阻塞赋值 vs 非阻塞赋值
    规则: 时序逻辑用 <=, 组合逻辑用 =
    项目: uart_tx.v — 全非阻塞赋值

2.2 状态机设计
    一段式: 状态+输出在同一 always 块
    二段式: 状态跳转 + 输出分离
    三段式: 状态跳转 + 次态逻辑 + 输出逻辑
    项目: uart_tx.v (一段式), eeprom_test.v (三段式), uii2c.v (三段式,7状态)

2.3 generate 语法
    用途: 批量实例化、条件编译
    三种形式: generate-for, generate-if, generate-case
    项目: uihdmitx.v 中有一处 generate-if 用法

2.4 parameter 参数化
    用途: 提高代码复用性
    项目: uart_tx.v #(CLK_FREQ, BAUD), uii2c.v #(CLK_DIV), key_debounce.v #(KEY_W)

三、L3 核心技能
----------------------------------------------------------------
3.1 时序约束
    3.1.1 时钟定义
        create_clock -name clk -period 20 -waveform {0 10} [get_ports {clk}]
        项目: alarm_clock/bd_prj/timing.sdc

    3.1.2 PLL 衍生时钟
        derive_pll_clocks → rename_clock
        项目: sdram_fdma_tpg/timing.sdc

    3.1.3 时钟分组
        set_clock_groups -exclusive -group {...}
        项目: sdram_fdma_tpg/timing.sdc (8个时钟域)

    3.1.4 引脚约束
        set_pin_assignment {signal} {LOCATION=Pxx; IOSTANDARD=LVCMOS33;}
        项目: alarm_clock/bd_prj/fpga_pin.adc

    3.1.5 STA 报告解读
        Setup Slack: 数据到达 vs 时钟需求 的时间差
        Hold Slack: 数据保持 vs 时钟边沿 的时间差
        项目: display_timing.rpt — 37.226ns setup slack

3.2 CDC 跨时钟域
    3.2.1 亚稳态原理
        触发器在捕获窗口内收到变化信号 → 输出不确定电平
        MTBF = e^(t/τ) / (Tw × fc × fd)

    3.2.2 单bit同步 — 两级同步器
        异步信号 → FF₁ → FF₂ → 稳定输出
        项目: key_debounce.v (key_r0 → key_r1)

    3.2.3 多bit同步 — 格雷码
        每次地址变化只翻转1bit → 跨域同步安全
        bin2gray: gray[i] = bin[i] ^ bin[i+1]
        gray2bin: bin[i] = bin[i+1] ^ gray[i]
        项目: wfifo.v 第250-328行

    3.2.4 多bit同步 — 异步FIFO
        格雷码传递地址 + BRAM存储数据 + 补码差判断满空
        项目: wfifo.v, rfifo.v

    3.2.5 多bit同步 — 握手协议
        发送: 放数据→翻转req
        接收: 两级同步req→检测变化→采样数据→翻转ack
        项目: (教程中未直接使用, CDC理论第四种解法)

    3.2.6 多bit同步 — 展宽+同步
        快域脉冲展宽 → 慢域可捕获
        (CDC理论第四种解法)

3.3 FIFO
    3.3.1 同步FIFO
        读写同时钟, 无CDC需求
        用途: 同域数据缓冲

    3.3.2 异步FIFO
        读写不同时钟, 格雷码跨域
        结构: 写地址计数器 + 读地址计数器 + 格雷码转换 + BRAM + 满空判断
        项目: wfifo.v (16bit→32bit), rfifo.v (32bit→16bit)

    3.3.3 FIFO 关键参数
        DATA_WIDTH_W/R: 写/读数据位宽
        ADDR_WIDTH_W/R: 写/读地址宽度
        AL_FULL_NUM / AL_EMPTY_NUM: 几乎满/几乎空阈值
        SHOW_AHEAD_EN: 首字预取模式

    3.3.4 FIFO 容量守恒
        写位宽 × 写深度 = 读位宽 × 读深度
        16bit × 2048 = 32bit × 1024 = 32,768 bits

3.4 接口协议
    3.4.1 UART
        协议: 1起始 + 8数据 + 1停止, LSB先发
        项目: uart_tx.v + uart_time_sender.v

    3.4.2 I2C
        协议: START → 地址+R/W → ACK → 数据 → ACK/NACK → STOP
        物理层: 开漏输出 + 上拉电阻, SCL时钟 + SDA数据
        项目: uii2c.v (7状态Master控制器) + eeprom_test.v

    3.4.3 SPI
        协议: CS↓ → SCLK边沿 → MOSI逐bit发送
        四种模式: CPOL(0/1) × CPHA(0/1)
        项目: ui_mspi_tx.v

    3.4.4 协议分层设计
        业务层(决定"做什么") → 协议层(决定"怎么发") → 物理层(硬件连接)
