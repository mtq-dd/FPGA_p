## 5. 可复用资产提炼

### 通用代码模块

- `key_debounce.v`：参数化按键消抖模块，`KEY_W`控制键数，`DELAY_TIME`控制消抖时间，可直接复用于任何按键输入场景
- `uart_tx.v`：参数化UART发送模块，`CLK_FREQ`和`BAUD`参数化，可直接复用于不同波特率需求
- POR电路模式（17位计数器+por_done信号）：可复用于任何需要上电复位的FPGA设计

### 通用约束模板

- 时钟约束模板：`create_clock -name clk -period 20 -waveform {0 10} [get_ports {clk}]`
- 引脚约束模板：`set_pin_assignment { signal } { LOCATION = Pxx; IOSTANDARD = LVCMOS33; }` + 按键/拨码的PULLTYPE配置

### 通用方法论

- 诊断固件法：写最小系统（diag_top.v，43行）逐一验证每个引脚，避免在复杂设计里定位bug
- 串口作为FPGA调试输出通道：每秒发送状态数据到PC串口终端，相当于软件开发的logcat
- LED心跳指示：用计数器分频产生多频率闪烁，指示不同模块运行状态
