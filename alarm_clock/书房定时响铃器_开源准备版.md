# 书房定时响铃器

## 项目简介

基于FPGA实现的数字闹钟，支持BCD计时、UART串口时间输出、蜂鸣器响铃、按键控制。低成本、零基础可复刻，适合有软件基础的开发者入门FPGA硬件开发。

## 硬件要求

- 开发板：米联客 MLKPAI-Fmin01（安路 EG4S20NG88）
- 物料清单：

| 器件 | 型号/说明 | 数量 | 成本 |
|------|----------|------|------|
| FPGA开发板 | MLKPAI-Fmin01 (EG4S20NG88) | 1 | 已有 |
| USB转UART模块 | CH340/CP2102 等 | 1 | ~5元 |
| 蜂鸣器 | 无源压电蜂鸣器（板载） | 1 | 0元（板载） |
| Micro-USB线 | 下载+供电 | 1 | ~5元 |

- 总硬件成本：约10元（不含开发板）

## 快速上手

1. 克隆本仓库
2. 安装 TD 5.6.4（安路官方IDE，免费）
3. 安装 JTAG 驱动：TD安装目录 `driver\dl-cable win10-x64\`，手动替换设备管理器中的驱动
4. 打开 TD，导入工程 `bd_prj\alarm_clock.al`
5. 确认器件型号为 `EG4S20NG88`
6. 按顺序执行：Read Design → Opt RTL → Opt Gate → Opt Placement → Opt Routing → Gen Bitstream
7. 连接 JTAG 下载器，Download → Add .bit → Run
8. USB转UART模块连接：TX接开发板P17，GND接GND，串口终端设115200-8-N-1
9. 按键操作：S1=复位，S2=切换模式（走时→设时钟→设闹钟），DIP1=手动进位，DIP2=手动增值

## 工程结构

```
alarm_clock/
├── alarm_clock_bd.v        # 顶层模块（含POR上电复位）
├── clock_bd.v              # 闹钟核心（BCD计时+闹钟+模式控制）
├── key_debounce.v          # 按键消抖模块（参数化）
├── uart_tx.v               # UART发送模块（参数化波特率）
├── uart_time_sender.v      # 时间→ASCII串格式化
├── diag_top.v              # 板级诊断固件（独立验证外设）
├── 上板总结_从仿真到硬件.md   # 踩坑心得+方法论
├── bd_prj/
│   ├── alarm_clock.al      # TD工程文件
│   ├── fpga_pin.adc        # 引脚约束
│   ├── timing.sdc          # 时钟约束
│   └── alarm_clock_Runs/   # 综合/布局/比特流输出
└── diag_prj/
    ├── fpga_pin.adc        # 诊断固件引脚约束
    └── timing.sdc          # 诊断固件时钟约束
```

## 核心功能

- [x] BCD格式24小时计时（时:分:秒）
- [x] UART串口每秒输出时间（"HH:MM:SS\r\n"，115200波特率）
- [x] 蜂鸣器闹钟响铃（763Hz方波驱动）
- [x] 2键控制：模式切换 + 手动进位/增值
- [x] 设置模式自动走位（每3秒自动切换当前编辑位）
- [x] 4LED状态指示（走时模式=心跳+mode，设置模式=pos位置指示）
- [x] POR上电复位（不依赖外部按键）
- [x] 按键消抖（1ms延时，参数化）
- [x] 板级诊断固件（独立验证所有外设引脚）

## 引脚分配

| 信号 | 引脚 | 说明 |
|------|------|------|
| clk | P34 | 系统时钟 25MHz |
| rst_n | P31 | 按键S1，低有效复位 |
| key[0] | P30 | 按键S2，模式切换 |
| key[1] | P45 | DIP1，手动进位/位选 |
| key[2] | P23 | DIP2，手动增值 |
| uart_tx | P17 | UART发送 |
| buzzer | P27 | 蜂鸣器 |
| led[0] | P51 | LED指示 |
| led[1] | P52 | LED指示 |
| led[2] | P57 | LED指示 |
| led[3] | P59 | LED指示 |

## 开发记录
